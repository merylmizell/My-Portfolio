Translation Accuracy Dashboard for AI Evaluation

Developed a dashboard to evaluate the quality of AI-generated translations of weather alerts using industry-standard metrics such as BLEU, chrF++, COMET, and Translation Edit Rate (TER). The system identified low-scoring translations and provided actionable insights for model refinement, including retranslation strategies and human-in-the-loop corrections. This tool aids in improving the accuracy of multilingual weather communications, ensuring accessibility for Limited English Proficiency (LEP) communities.

Translation Log Dashboard for Real-Time Latency

Designed and implemented a dashboard to measure and analyze the time required to complete translations of critical weather alerts. Leveraged AWS S3 Bucket and DynamoDB to collect and process real-time data. Insights from latency metrics were used to identify process bottlenecks, optimize AI model efficiency, and compare cloud-based versus local translation performance. This tool enables the National Weather Service to monitor and reduce delays in delivering life-saving weather alerts.

Stock Recommendation System Based on ESG Scores and LSTM Model

This project develops a stock recommendation system that combines historical stock data and Environmental, Social, and Governance (ESG) metrics to assist ethical and sustainable investment decisions. Using the Long Short-Term Memory (LSTM) model, the system predicts stock prices while incorporating ESG scores to evaluate companies on financial and sustainability performance. Data preprocessing included merging datasets based on ticker symbols, handling missing values through imputation or removal, and feature scaling with MinMaxScaler to optimize LSTM training. Performance metrics, such as Mean Squared Error (MSE) and Sharpe ratio, validated the system's effectiveness, highlighting correlations between higher ESG scores and improved financial performance.

Neo4j-Based Knowledge Graph Chat for Real-Time Water Quality Awareness

Developed an interactive chatbot system leveraging Neo4j to provide real-time insights into water quality metrics collected by Blue CoLab's monitoring stations. Integrated real-time data on pH, temperature, turbidity, salinity, and dissolved oxygen into a graph database, enabling sophisticated exploration of environmental relationships. The chatbot, built with Flask and enhanced by natural language processing (NLP), translates user queries into actionable insights using Cypher queries. This project fosters public understanding of water safety by offering intuitive access to real-time environmental data, empowering stakeholders with critical knowledge for informed decision-making.

Island Escape Game

Developed a survival game using Unity and C#, titled "Island Escape," where players navigate an island to collect barrels of supplies while avoiding obstacles. Once all barrels are gathered, players proceed to the shore to access the escape raft. The game features intuitive controls: 'A' for left, 'D' for right, 'W' for backward, 'S' for forward, and left mouse click to attack.
